获取取消原因

接口简介

调用取消订单接口时，需要传递取消原因ID，通过此接口获取订单取消理由列表

接口调用请求说明，URL地址：/api/order/cancel/reasons。

请求参数说明

无

注：当没有业务参数的时候, body需要赋值为空字符串,即body:""。

返回结果

返回JSON格式，以下为参数说明：

参数名称 类型 描述
statusCode Integer 响应状态，成功为"200"，失败为"300"
message String 如果错误则显示错误原因
data Object 响应结果，JSON对象，详见具体的接口描述

data业务参数：

参数名称 类型 描述
id Integer 理由编号
reason String 取消理由
