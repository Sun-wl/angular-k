门店详情

接口简介

查询门店详情接口,接口URL地址：/api/shop/detail

请求参数说明

名称 类型 是否必传 描述
origin_shop_id String 是 门店编码
请求参数示例 
                {
                    "origin_shop_id": "shop001",
                }

返回结果

返回JSON格式，以下为参数说明：

参数名称 类型 描述
statusCode Integer 响应状态，成功为"200"，失败为"300"
message String 如果错误则显示错误原因
data Object 响应结果，JSON对象，详见具体的接口描述

data业务参数：

参数名称 类型 描述
origin_shop_id String 门店编码
station_name String 门店名称
city_code String 城市的code（查看各城市对应的code值)
area_name String 区域名称
station_address String 门店地址
station_address_details String 门店详细地址
lng Double 门店经度
lat Double 门店纬度
contact String 联系人姓名
mobile String 联系人电话
status Integer 门店状态（2-门店激活，1-停用）
正确的响应结果（JSON格式） 
                {
                    "data": {
                        "station_name": "shop001",
                        "area_name": "闵行区",
                        "station_address": "明翰教育(上中西路)",
                        "station_address_details": "201室",
                        "city_code": "上海",
                        "contact": "shop001",
                        "origin_shop_id": "shop001",
                        "lng": 121.407631,
                        "mobile": "13810001001",
                        "lat": 31.129922,
                        "status": 0
                      },
                    "statusCode": 200,
                    "message": "成功"
                }
失败的响应结果（JSON格式） 
                {
                    "statusCode": 300,
                    "message": "门店不存在"
                }
