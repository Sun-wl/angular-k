新增门店

接口简介

批量新增门店接口,接口URL地址：/api/shop/add。

1. 门店编码可自定义，但必须唯一，若不填写，则系统自动生成。发单时用于确认发单门店

2. 该接口为批量接口,业务参数为数组

请求参数说明

名称 类型 是否必传 描述
origin_shop_id String 否 门店编码,可自定义,但必须唯一;若不填写,则系统自动生成
station_name String 是 门店名称
city_code String 是 所在城市的code（查看各城市对应的code值)
area_name String 是 区域名称(如,浦东新区)
station_address String 是 门店地址
station_address_details String 否 门店详细地址/门牌号
lng Double 是 门店经度
lat Double 是 门店纬度
contact String 是 联系人姓名
mobile String 是 联系人电话

返回结果

返回JSON格式，以下为参数说明：

参数名称 类型 描述
statusCode Integer 响应状态，成功为"200"，失败为"300"
message String 如果错误则显示错误原因
data Object 响应结果，JSON对象，详见具体的接口描述
参数名称 类型 描述
success Integer 成功导入门店的数量
successList List 成功导入的门店
failedList List 导入失败门店编号以及原因
