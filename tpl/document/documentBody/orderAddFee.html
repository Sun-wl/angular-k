增加小费

接口简介

可以对待接单状态的订单增加小费。需要注意：订单的小费，以最新一次加小费动作的金额为准，故下一次增加小费额必须大于上一次小费额。

接口调用URL地址：/api/order/addTip。

请求参数说明

名称 类型 是否必传 描述
order_id String 是 第三方订单编号
tipMoney Integer 是 小费金额 (单位 : 分)

返回结果

返回JSON格式，以下为参数说明：

参数名称 类型 描述
statusCode Integer 响应状态，成功为"200"，失败为"300"
message String 如果错误则显示错误原因
data Object 响应结果，JSON对象，详见具体的接口描述
