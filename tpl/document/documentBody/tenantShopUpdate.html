编辑门店

接口简介

更新门店信息接口,接口URL地址：/api/shop/update

门店编码是必传参数。其他参数，需要更新则传，且不能为空。

请求参数说明

名称 类型 是否必传 描述
origin_shop_id String 是 门店编码
station_name String 否 门店名称
city_code String 否 城市的code（查看各城市对应的code值)
area_name String 否 区域名称(如,浦东新区)
station_address String 否 门店地址
station_address_details String 否 门店详细地址
lng Double 否 门店经度
lat Double 否 门店纬度
contact String 否 联系人姓名
mobile String 否 联系人电话
status Integer 否 门店状态（2-门店激活，1-停用）
请求参数示例 
                {
                    "origin_shop_id": "13810001001",
                    "station_name": "更新门店"
                }

返回结果

返回JSON格式，以下为参数说明：

参数名称 类型 描述
statusCode Integer 响应状态，成功为"200"，失败为"300"
message String 如果错误则显示错误原因
data Object 响应结果，JSON对象，详见具体的接口描述
正确的响应结果（JSON格式） 
                {
                    "statusCode": 200,
                    "message": "成功"
                }
失败的响应结果（JSON格式） 
                {
                    "message": 300,
                    "message": "没有可以更新的参数,请重新核对"
                }
