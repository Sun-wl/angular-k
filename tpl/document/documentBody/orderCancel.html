取消订单(线上环境)

接口简介

在订单待接单或待取货情况下，调用此接口可取消订单。

接口调用请求说明，URL地址：/api/order/formalCancel

请求参数说明

名称 类型 是否必传 描述
order_id String 是 第三方订单编号
cancelReasonId Integer 是 取消原因ID
cancelReason String 是 取消原因(当取消原因ID为其他时，此字段必填)

返回结果

返回JSON格式，以下为参数说明：

参数名称 类型 描述
statusCode Integer 响应状态，成功为"200"，失败为"300"
message String 如果错误则显示错误原因
data Object 响应结果，JSON对象，详见具体的接口描述
