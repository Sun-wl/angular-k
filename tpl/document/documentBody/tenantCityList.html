获取城市信息

接口简介

通过接口，获取城市信息列表。

接口调用URL地址：/api/cityCode/list。

请求参数说明

无

注：当没有业务参数的时候, body需要赋值为空字符串,即body:""。

返回结果

返回json格式，以下为参数说明：

参数名称 类型 描述
statusCode Integer 响应状态，成功为"200"，失败为"300"
message String 如果错误则显示错误原因
data Object 响应结果，JSON对象，详见具体的接口描述

data业务参数：

参数名称 类型 描述
name String 城市名称
code String 城市ID
