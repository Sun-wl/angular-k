骑手接单(仅在测试环境供调试使用)

接口简介

在测试环境中，可调用此接口接受订单，检测回调。

注：接口仅限于测试环境调试使用，且触发回调URL成功后，接口直接返回成功

接口调用URL地址：/api/order/test/receiving。

请求参数说明

名称 类型 必传 描述
origin_id String 是 第三方订单id

返回结果

返回JSON格式，以下为参数说明：

参数名称 类型 描述
statusCode Integer 响应状态，成功为"200"，失败为"300"
message String 如果错误则显示错误原因
data Object 响应结果，JSON对象，详见具体的接口描述
